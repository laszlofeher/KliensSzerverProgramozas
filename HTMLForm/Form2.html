Form Basic

A form tag method attributumában két értéket lehet megadni, GET/POST, ebben a példában a POST értéket használtuk. Az input tag type attributuma sok féle értéket vehet fel, az egyik leggyakrabban használt a text, ha ezt elrontjuk és mást írunk bele pl.: kiskutya, akkor is alapértelmezetten a böngésző a TextBox-ot készít. Az input tag name attributumába írt azonosítót a szerveroldali programnyelvek használják kulcsként Az input tag id attributumába írt azonosítót a kliens oldali javascript vagy a CSS használja és a taget egyedileg azonosítja. Az input tag value attributumába írt értéke megjelenik az input mezőben, ehhez kapcsoloódik a HTML5 ben a placeholder attributum, ami a HTML5 szabvánnyal érlezett, és egy halványabb szöveget tesz a TextBox-ba, továbbá ha bele kattintunk a TextBox-ba, akkor eltünik a felirat.
